# Aufgabenverteilung

Dieses Dokument beschreibt die wichtigsten Aufgabenbereiche im Projekt.

## Projektstruktur

- Aufbau der Ordnerstruktur
- Trennung von Source-Code, Tests, Daten und Dokumentation
- Erstellung von README, Dockerfile und docker-compose.yml

## Backend

- Erstellung eines Flask-Servers
- Bereitstellung von simulierten Energiedaten
- API-Endpunkte für aktuelle und historische Daten

## Frontend

- Erstellung eines Streamlit-Dashboards
- Anzeige von Kennzahlen
- Anzeige von Tabellen und Diagrammen

## Datenverarbeitung

- Laden von CSV-Dateien
- Prüfung und Bereinigung der Daten
- Berechnung von Bilanz und Kalenderwoche

## Berechnungen

- Berechnung von Gesamtverbrauch
- Berechnung von Gesamterzeugung
- Berechnung von Eigenversorgung
- Berechnung von erspartem Geld
- Bestimmung des Gesamtstatus

## Tests und Qualität

- Unit-Tests mit pytest
- Docker-Unterstützung
- CI/CD-Vorbereitung mit GitHub Actions
- Formatierung mit Black
- Codeprüfung mit Ruff
